# Hermes Skills

Custom skills for Hermes Agent synced across work/home machines.

## Usage

Add this repo as a skills source:
```
hermes skills tap add dbsuperss-ops/hermes-skills
```

## Structure

Each skill goes in its own directory:
```
skills/
  my-skill/
    SKILL.md       # Skill definition
    references/    # Optional reference docs
    scripts/       # Optional scripts
    templates/     # Optional templates
```
